Here are some of my projects

Room Finder

Room Finder is a web application that allows users to list their properties for rent or sale and other users to find properties for them to rent or buy. The app is made using React JS as main framework and Firebase for backend. It is also fully responsive.

HowzTheWeather

HowzTheWeather is a web app that shows the weather of a city. Shows additional information like pressure, wind speed, humidity, etc. Uses API data from Open Weather Map. Made using React JS.

Interactive NumPad

Interactive numpad is a interactive 3D design made using Spline. We can type the numbers in our keyboard or the numpad and it shows the animation on the screen.
